## 支援架構

1. .net 6.0

## 功能簡介

封裝常用語法，精簡化

自用、長期維護、緩慢更新

## 可以使用的方法
| 方法 | 備註 |
| :--- | :--- |
| ToUnixTimestamp |  將UTC DateTime轉換為Unix時間戳 |
| FromUnixTimestamp |   將Unix時間戳轉換為UTC DateTime |
| GetDescription |  從枚舉常數中獲取描述文本 |
| ErrorMessageException |  自訂意錯誤訊息 |
| ToJsonString |  將物件序列化為JSON字符串   |
| FromJsonString |  將JSON字符串反序列化為指定的類型 |
